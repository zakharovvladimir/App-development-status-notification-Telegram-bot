# Status notification Telegram bot

### Description
App development status notification Telegram bot

### How to use 
add to your .env:
```bash
PRACTICUM_TOKEN = ''
TELEGRAM_TOKEN = ''
TELEGRAM_CHAT_ID = ''
```

RETRY_PERIOD = 600
ENDPOINT = 'https://practicum.yandex.ru/api/user_api/homework_statuses/'
HEADERS = {'Authorization': f'OAuth {PRACTICUM_TOKEN}'}
